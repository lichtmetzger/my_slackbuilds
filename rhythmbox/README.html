Rhythmbox for Slackware64-current

Either add this to slapt-getrc:

SOURCE=https://schmatzler.de/my_slackbuilds/rhythmbox/:OFFICIAL

Install the packages:

slapt-get --install avahi BeautifulSoup bottle brasero clutter clutter-gst clutter-gtk cogl dbus-python3 gst-plugins-bad gst-plugins-ugly hyphen json-glib lame libburn libdaemon libdmapsharing libisofs libmodplug libmusicbrainz3 libpeas lxml mx py3cairo pygobject3-python3 python3 rhythmbox rhythmbox-plugin-alternative-toolbar rhythmbox-plugin-ampache rhythmbox-plugin-android-remote rhythmbox-plugin-equalizer rhythmbox-plugin-hide rhythmbox-plugin-lastfmplaycount rhythmbox-plugin-llyrics rhythmbox-plugin-opencontainingfolder rhythmbox-plugin-radio-browser rhythmbox-plugin-ratingfilters rhythmbox-plugin-rbtempo rhythmbox-plugin-rhythmremote rhythmbox-plugin-rhythmweb rhythmbox-plugin-rtr rhythmbox-plugin-spectrum rhythmbox-plugin-tray-icon rhythmbox-plugin-wikipedia-search rhythmbox-plugin-youtube-search soundtouch totem-pl-parser vala
---

Or edit /etc/slackpkg/slackpkgplus.conf and make sure you alter REPOPLUS and add MIRRORPLUS to create a new repository 'rhythmbox':

REPOPLUS=( slackpkgplus rhythmbox )
MIRRORPLUS['rhythmbox']=https://schmatzler.de/my_slackbuilds/rhythmbox/

Then install it:

slackpkg install rhythmbox

If you want to use the dark theme, edit /etc/gtk-3.0/settings.ini and add

[Settings]
gtk-application-prefer-dark-theme = true

For Avahi to work, you need to create a new user:

groupadd -g 214 avahi
useradd -u 214 -g 214 -c "Avahi" -d /dev/null -s /bin/false avahi
